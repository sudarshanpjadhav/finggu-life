🚀 5-minute Setup Guide
1
Create a free Supabase account
Go to supabase.com → click Start for free → sign up with Google or email (it's free).
2
Create a new project
Click New Project → name it finggu-life → choose region South Asia (ap-south-1) → set a database password → click Create Project. Wait ~2 minutes.
3
Run the database setup script
In your Supabase dashboard → click SQL Editor (left sidebar) → click New Query → copy the schema SQL from this file on GitHub → paste it → click Run.
4
Get your Project URL and Anon Key
In Supabase dashboard → click Settings (⚙️ icon, left sidebar) → click API → copy the two values shown below:
